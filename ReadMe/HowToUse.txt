- 这是一个倒立摆模拟器，主窗口分为5个区域

	- 上半部分是显示区，
		- 红色直线即为模拟的摆杆
		- 蓝色网格便于看观察杆角度，其中水平的粗线看作滑车轨道，水平粗线和垂直粗线的交点为原点
		- 滑车忽略没有画（滑车质量为摆杆100倍）
	
	- 下半部分分成四个区域
		- 左上：控制按钮区：
			- 启动模拟、复位摆杆、暂停模拟、进行自动复位设置
			- 如果开启自动复位，当模拟出异常数据时可以让摆杆自动复位，避免数值过大使程序崩溃
		
		- 左下：PID设置区：
			- 提供了四个位置式pid，点击使能来让各个环参与pid运算，也可以只使用一部分pid
			- out显示了该pid环的输出量；两个位置环的期望值都恒定为0（在原点竖直）
		
		- 右上：扰动设置区：
			- 在输入框中输入0~99的数字，点击四个按键，向摆杆系统中输入干扰
		
		- 右下：数据查看区：
			- 这里顶部有一个推子，是用来设置复位角度所用。当模拟暂停时，可以拉动推子进行设置。
            - 点击复位后，摆杆自动回到原点，并恢复为推子设定的角度，重新开始模拟
			- 下面的六个数据框实时显示摆杆状态

- 注意：
	- 如果要进行PID模拟，强烈建议开启自动复位，否则程序很可能崩溃
	- 位移位置环期望值为0，请在原点附近使能此环

- 在菜单栏点选“文件选择器”，可以进行配置参数的加载和保存
	- 快捷键：CTRL+F
	- 默认保存路径为setting文件夹
	- 配置文件为.txt文件
	- 点选文件名时，如果是.txt文件，可以在右侧预览文件内容，点击 "加载" 按钮加载参数
